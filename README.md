# Simplify

A minimal, Notion-inspired productivity dashboard focused on clarity, calm UI, and content-first design.

## Tech Stack
- HTML
- SCSS (CSS Grid, design tokens)
- Vanilla JavaScript

## Highlights
- Sidebar-based layout (no traditional navbar)
- Scrollable banner / page cover
- Clean dark theme with semantic tokens
- Content-first, distraction-free UI

## Status
Work in progress. Features and UI are being added incrementally.
